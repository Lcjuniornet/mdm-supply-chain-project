# Learning Journey - Projeto MDM Supply Chain

## Contexto

Este documento registra o processo de aprendizado durante o projeto.
Para documentação executiva, veja README.md principal.

## Semana 1: Fundamentos Python + Pandas

### Desafios Enfrentados:
- Jupyter Notebook: Células fora de ordem (NameError)
- Solução: Restart & Run All workflow
- Aprendizado: Ambiente reproduzível é crítico

### Skills Adquiridas:
- pandas groupby, duplicated(), value_counts()
- matplotlib + seaborn para visualizações
- Cálculo de score ponderado

## Semana 2: SQL + Análises Complementares

### Desafios Enfrentados:
- Window Functions: conceito novo
- Solução: Estudar ROW_NUMBER + PARTITION BY
- Aprendizado: SQL analítico ≠ SQL transacional

### Skills Adquiridas:
- CTEs (Common Table Expressions)
- Window Functions (ROW_NUMBER, RANK, PARTITION BY)
- Score de qualidade em SQL puro

## Semana 4: Power BI + DAX

### Desafios Enfrentados:
- DAX context (row vs filter context)
- Solução: Estudar documentação + prática
- Aprendizado: Medidas ≠ Colunas calculadas

### Skills Adquiridas:
- Modelagem star schema
- Medidas DAX complexas (CALCULATE, FILTER)
- Otimização performance (query folding)

## Estatísticas de Aprendizado

- **Horas investidas:** ~80 horas (49 dias × 1,5h/dia)

1. **Projeto > Curso:** Aprendi 10× mais fazendo que assistindo
2. **Debugging = Aprendizado:** Cada erro resolvido solidifica conhecimento
3. **Documentar = Consolidar:** Escrever SKILL.md fixou conceitos
4. **Portfolio >> Certificado:** Recrutadores querem ver CÓDIGO, não PDF
